# Ui-Exercise

Gmail responsive UI replica code challenge built with Angular.

## Dependencies

Use the following command to download the project dependencies:

```bash
npm install
```

## Run

Do a:

```bash
ng serve
```

Then go to `http://localhost:4200` on your browser.

## Tests

Unit tested some components with Karma/Jasmine.

```bash
npm run test
```

## Lint

```bash
npm run lint
```
